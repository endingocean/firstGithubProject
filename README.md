# AI 新闻热点

一个简洁漂亮的 AI 新闻展示静态网页，展示每日最热门的 AI 资讯。

## 功能特点

- 🎨 深色主题，现代化 UI 设计
- 📅 支持日期导航，查看历史新闻
- 🔥 热度指数展示
- 📱 响应式设计，适配各种设备
- ⚡ 纯静态网页，无需后端

## 在线访问

网站已部署到 GitHub Pages：https://endingocean.github.io/firstGithubProject/

## 技术栈

- HTML5
- CSS3（内联样式）
- JavaScript（原生 ES6+）
- 无需构建工具，直接浏览器运行

## 本地运行

直接在浏览器中打开 `index.html` 文件即可。

## 部署

项目配置了 GitHub Actions，每次推送到 `static-site` 分支会自动部署到 GitHub Pages。

## 数据来源

新闻数据为示例数据，存储在 JavaScript 中。如需更新，编辑 `index.html` 中的 `newsData` 对象。
